# is-date-within-range
Check if a given date is within a period of a given date, returns `true` or `false`

#### NPM
```
  $ npm install js-cookie --save
```

Read cookie:

```javascript
if(dateCheck("1999-01-01","2018-06-21","2019-06-06"))
    console.log("Availed");
else
    console.log("Not Availed");
```
